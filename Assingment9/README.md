# Python Assignments 1

This repository contains Python assignments for class homework, featuring practical exercises to enhance understanding and proficiency in Python programming.

---

## Questions

### 1. Display Function
Write a program which contains one function named `Display()` that prints "Jay Ganesh" on console.

**Example:**
```
Output: Jay Ganesh
```

---

### 2. Compare Two Numbers
Write a program which contains one function `ChkGreater()` that accepts two numbers and prints the greater number.

**Example:**
```
Input: 10 20
Output: 20 is greater
```

---

### 3. Square of a Number
Write a program which accepts one number and prints the square of that number.

**Example:**
```
Input: 5
Output: 25
```

---

### 4. Cube of a Number
Write a program which accepts one number and prints the cube of that number.

**Example:**
```
Input: 3
Output: 27
```

---

### 5. Divisibility Check
Write a program which accepts one number and checks whether it is divisible by both 3 and 5.

**Example:**
```
Input: 15
Output: Divisible by 3 and 5
```
